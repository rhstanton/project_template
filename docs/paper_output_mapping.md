## Quick Command Reference

### Replicate All Main Results
```bash
make all              # Build all artifacts (~5 minutes)
```

### Replicate by Category
```bash
# If you organize analyses by topic:
make housing          # All housing analyses
make mortgage         # All mortgage analyses
make income           # All income analyses
```

### Replicate Individual Key Exhibits
```bash
# Main effects
make price_base       # Figure 1, Table 1
make remodel_base     # Figure 2, Table 2

# Robustness
make price_robust     # Robustness checks
make remodel_robust   # Alternative specifications

# Heterogeneity
make price_het        # Heterogeneity analysis
```

## Published Paper Directory

After running `make publish`, outputs are copied to `paper/`:

```
paper/
├── figures/
│   ├── price_base.pdf          # → Paper Figure 1
│   └── remodel_base.pdf        # → Paper Figure 2
├── tables/
│   ├── price_base.tex          # → Paper Table 1
│   └── remodel_base.tex        # → Paper Table 2
└── provenance.yml              # Publication metadata
```

**Note:** `paper/` is typically a separate git repository for the manuscript.

```latex
\begin{table}[htbp]

  \label{tab:price_did}

\end{table}
```

## Exhibit Descriptions

### Figure 1: House Price Effects (`price_base.pdf`)

**What it shows:**

- Event study plot of house prices relative to fire event
- X-axis: Years relative to fire (-10 to +10)
- Y-axis: Treatment effect on log price
- Includes 95% confidence intervals

**Key result:** ~25% price drop in year 0, recovering over subsequent years

**Generated by:** `build_price_base.py`

**Data:** `data/housing_panel.csv`

---

### Figure 2: Remodeling Effects (`remodel_base.pdf`)

**What it shows:**

- Event study plot of remodeling activity
- X-axis: Years relative to fire
- Y-axis: Treatment effect on square footage

**Key result:** Spike in remodeling immediately after fire

**Generated by:** `build_remodel_base.py`

**Data:** `data/housing_panel.csv`

---

### Table 1: Price DiD Coefficients (`price_base.tex`)

**What it shows:**

- Difference-in-differences regression results
- Treatment effects at t=0, t=1, t=2, etc.
- Standard errors, R-squared, observation counts

**Key result:** Significant negative effect at t=0

**Generated by:** `build_price_base.py`

---

### Table 2: Remodeling DiD Coefficients (`remodel_base.tex`)

**What it shows:**

- DiD regression results for remodeling
- Treatment effects over time
- Statistical significance indicators

**Key result:** Positive effect on remodeling activity

**Generated by:** `build_remodel_base.py`

## Notes

### Static Figures

If you have hand-drawn figures or maps:

- Store in `paper/figures/` (not generated by code)
- Document in this file as "static/pre-generated"
- Example: `Figure 0: Study area map (static)`

### Multi-Panel Figures

If a single paper figure combines multiple outputs:
```
| Figure 3 | `fig:combined` | Combined analysis | Panel A: `price_base.pdf`<br>Panel B: `price_robust.pdf` | `make price_base price_robust` |
```

### Summary Statistics Tables

Many analyses generate both:

- Main regression table (e.g., `price_base.tex`)
- Summary statistics (e.g., `price_base_sumstats.tex`)

Document both if referenced in paper.

### Numerical Values in Text

If paper text cites specific numbers:
```
| In-text value | Description | Source | How to find |
|---------------|-------------|--------|-------------|
| "25% price drop" | Price effect at t=0 | Table 1, row 3 | grep "t=0" output/tables/price_base.tex |
```

## Updating This Mapping

**When you add a new analysis:**

1. Add script: `build_your_analysis.py`
2. Update `ARTIFACTS` in `Makefile`
3. Add entry to this file
4. Run: `make your_analysis`
5. Commit: `git add -A && git commit -m "Add your_analysis"`

**Template for new entries:**
```
| Figure X | `fig:label` | Description | `output/figures/name.pdf` | `make name` |
```

## Verification

**To verify all outputs exist:**
```bash
make test-outputs
```

**To verify a specific output hasn't changed:**
```bash
sha256sum output/figures/price_base.pdf
# Compare with hash in output/provenance/price_base.yml
```

**To reproduce exact paper version:**
```bash
git checkout [paper-submission-tag]
make clean
make all
# Outputs should match checksums in paper/provenance.yml
```
